# 🛒 Amazon Review Sentiment Analysis

A Natural Language Processing (NLP) project that analyzes Amazon product reviews and predicts whether a review expresses positive or negative sentiment using machine learning techniques.

---

## 📌 Project Overview

Customer reviews contain valuable insights about products. This project processes raw Amazon product reviews, cleans and transforms the text data, and applies machine learning models to classify sentiments.

The project follows an end-to-end NLP pipeline including data preprocessing, feature engineering, model training, and evaluation. It can be extended to production using FastAPI.

---

## 🧠 Problem Statement

Given a product review written by a customer, classify the sentiment of the review as:
- Positive
- Negative
